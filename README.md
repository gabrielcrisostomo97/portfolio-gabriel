        ## 🚀 Como rodar localmente
        1. Clone este repositório:
        ```bash
        git clone https://github.com/gabrielcrisostomo97/portfolio-gabriel.git
        Entre na pasta:

        cd portfolio-gabriel


        Instale as dependências:

        npm install


        Inicie o servidor:

        npm run dev
